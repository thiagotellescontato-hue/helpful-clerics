# Helpful Armorers

Helpful Armorers is a Fabric mod for Minecraft 1.21.1 that lets armorer villagers repair damaged iron golems when the area is safe.

## Features

- Armorer villagers search for nearby damaged iron golems.
- Armorers move close to the golem, hold an iron ingot, and repair it.
- Armorers stop repairing when hostile mobs are nearby, at night, or after taking damage.

## Requirements

- Minecraft 1.21.1
- Fabric Loader
- Fabric API
- Java 21
